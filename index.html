Inside the Book ...

A quick glance at the topics you'll learn

This chapter is about web designing and its concept, in this chapter you'll learn how create a template for your or your client website. You'll learn about UI/UX design that can architect the base of a web site project.

This is the begining of web desinging chapter. HTML is a language that you can start your web designing journey, in this chapter you'll learn the HTML tags, attributes and etc. HTML is stand for Hyper Text Markup Languge that you can structure your website.

CSS stands for Cascading Style Sheet that you can design your pages.By CSS you can chosse an element display where and how it will display.In this chapter you'll learn how to style your element which you created by HTML. HTML and CSS are necessary for designing a web page.

JavaScript is a scripting language that you can create intersting and dynamic web pages, desktop application or even mobile application. In this chapter you'll learn how to use JavaScript in your web projects to make your web page become interactive for your user. When your web pages are interactive user will not tired.

Big systems are need a place to store their data. for example Google, Facebook, Twitter and ... are have millions of users that everu user have an acount and their data should sore in a place that is why these big system use database. in this chapter you'll learn how to store data in databases and also you'll learn about basics of databases.
